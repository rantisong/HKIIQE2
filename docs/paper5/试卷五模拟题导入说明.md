# 试卷五模拟题导入说明

已从 `试卷五：mock.pdf` 解析生成 **620 道题目**，输出文件：`docs/试卷五mock_import.json`。

## 导入方式

使用云函数 **paper_importFromJson**：

### 方式一：云存储 + 云函数（推荐）

1. 微信开发者工具 → 云开发 → 云存储，上传 `docs/试卷五mock_import.json`。
2. 云函数 → `paper_importFromJson` → 云端测试，参数：`{ "fileId": "该文件的 File ID" }`。

### 方式二：直接传 content

在云函数测试参数中传入 `"content": <试卷五mock_import.json 的完整内容>`。

## 导入后

- 模拟题库「卷五」会出现在练习页，选择卷五进入试卷选择页后可进行「随机抽题练习」（每次从 620 题中随机抽 80 题）。
- 若已有 subjectId 为 `05` 的模拟卷，本次导入会**覆盖**该记录。

## 重新解析 PDF

```bash
pip install pypdf
python3 scripts/parse_mock_pdf.py "/path/to/试卷五：mock.pdf" 5 docs/试卷五mock_import.json
```
