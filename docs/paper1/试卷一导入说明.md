# 试卷一模拟题导入说明

已从 `试卷一mock.pdf` 解析生成 **769 道题目**，输出文件：`docs/试卷一mock_import.json`。

## 方式一：云存储 + 云函数（推荐，适合大文件）

1. 打开**微信开发者工具** → 云开发 → **云存储**。
2. 上传 `docs/试卷一mock_import.json` 到存储，路径可设为：`papers/试卷一mock_import.json`。
3. 上传完成后，在文件列表中点击该文件，复制 **File ID**。
4. 打开 **云函数** → 选择 `paper_importFromJson` → **云端测试**。
5. 输入参数（将 `你的FileID` 替换为实际 File ID）：
   ```json
   { "fileId": "你的FileID" }
   ```
6. 执行后若返回 `success: true`，即导入成功。

## 方式二：直接传 content（适合小体积 JSON）

在云函数 `paper_importFromJson` 的测试参数中传入：

```json
{
  "content": { 这里粘贴 试卷一mock_import.json 的完整内容 }
}
```

注意：请求体过大会受云函数入参大小限制，约 465KB 时建议用方式一。

## 导入后

- 模拟题库「卷一」会出现在练习页，选择卷一进入试卷选择页后可进行「随机抽题练习」（每次从 769 题中随机抽 75 题）。
- 若之前已有 subjectId 为 `01` 的模拟卷，本次导入会**覆盖**该记录。
