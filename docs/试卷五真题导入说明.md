# 试卷五真题导入说明

已从 5 份试卷五真题 PDF 解析生成可导入 JSON，均带 **subjectId: "05"**，导入后只会在**卷五**的「考试真题」列表中展示。

## 生成的文件（docs/）

| 文件 | 题目数 |
|------|--------|
| real_05_2025年10月真题.json | 98 |
| real_05_2025年11月真题.json | 93 |
| real_05_2025年12月真题.json | 96 |
| real_05_2026年1月真题.json | 65 |
| real_05_2026年2月真题.json | 67 |

## 导入方式

使用云函数 **paper_importFromJson**，对**每个 JSON 文件**各执行一次导入。

### 方式一：云存储 + 云函数（推荐）

1. 云开发 → 云存储，将上述 5 个 JSON 上传。
2. 云函数 → `paper_importFromJson` → 云端测试，对每个文件执行：`{ "fileId": "该文件的 File ID" }`。

### 方式二：直接传 content

在云函数测试参数中传入 `"content": <对应 JSON 的完整内容>`，每个文件执行一次。

## 导入后

- 练习页选择**卷五**进入试卷选择页后，「考试真题」列表会显示这 5 份真题。
- 其他科目的真题不会出现在卷五列表（按 subjectId 筛选）。

## 重新解析 PDF

```bash
python3 scripts/parse_real_paper_pdf.py "/path/to/試卷五：2025年10月真题.pdf" 5 docs/real_05_2025年10月真题.json
```

科目 5 = 卷五（投资相连长期保险）。
