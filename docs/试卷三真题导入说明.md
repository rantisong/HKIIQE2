# 试卷三真题导入说明

已从 5 份试卷三真题 PDF 解析生成可导入 JSON，均带 **subjectId: "03"**，导入后只会在**卷三**的「考试真题」列表中展示。

| 文件 | 题目数 |
|------|--------|
| real_03_2025年10月真题.json | 88 |
| real_03_2025年11月真题.json | 86 |
| real_03_2025年12月真题.json | 85 |
| real_03_2026年1月真题.json | 67 |
| real_03_2026年2月真题.json | 61 |

## 导入方式

使用云函数 **paper_importFromJson**，对**每个 JSON 文件**各执行一次导入。

### 方式一：云存储 + 云函数（推荐）

1. 微信开发者工具 → 云开发 → 云存储，将上述 5 个 JSON 上传（如放到 `papers/试卷三/` 下）。
2. 云函数 → `paper_importFromJson` → 云端测试，对每个文件执行一次，参数：`{ "fileId": "该文件的 File ID" }`。

### 方式二：直接传 content

在云函数测试参数中传入 `"content": <对应 JSON 的完整内容>`，每个文件执行一次。

## 导入后

- 用户在练习页选择**卷三**进入试卷选择页后，「考试真题」列表会显示：2025年10月、2025年11月、2025年12月、2026年1月、2026年2月真题。
- 试卷一的真题不会出现在卷三列表（按 subjectId 筛选）。
