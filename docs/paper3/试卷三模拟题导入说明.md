# 试卷三模拟题导入说明

已从 `試卷三：mock.pdf` 解析生成 **728 道题目**，输出文件：`docs/试卷三mock_import.json`。

## 导入方式

与试卷一相同，使用云函数 **paper_importFromJson**：

### 方式一：云存储 + 云函数（推荐，文件较大）

1. 微信开发者工具 → 云开发 → 云存储，上传 `docs/试卷三mock_import.json`（如路径 `papers/试卷三mock_import.json`）。
2. 云函数 → `paper_importFromJson` → 云端测试，参数：
   ```json
   { "fileId": "云存储中该文件的 File ID" }
   ```

### 方式二：直接传 content

若 JSON 体积在云函数入参限制内，可在测试参数中传入：
```json
{
  "content": <粘贴 试卷三mock_import.json 的完整内容>
}
```

## 导入后

- 模拟题库「卷三」会出现在练习页，选择卷三进入试卷选择页后可进行「随机抽题练习」（每次从 728 题中随机抽 50 题）。
- 若之前已有 subjectId 为 `03` 的模拟卷，本次导入会**覆盖**该记录。

## 重新解析 PDF

若需从 PDF 重新生成 JSON：

```bash
pip install pypdf
python3 scripts/parse_mock_pdf.py "/path/to/試卷三：mock.pdf" 3 docs/试卷三mock_import.json
```

参数：`<PDF路径>` `[科目1-5]` `[输出路径]`，科目 3 对应卷三（长期保险）。
