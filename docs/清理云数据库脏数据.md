# 清理微信云数据库脏数据

系统已改为使用 **mock_bank**（模拟题库）、**real_papers**（真题）两个集合，旧的 **papers** 集合不再使用，其中的数据视为脏数据，可安全清空。

## 操作步骤

1. 打开**微信开发者工具** → **云开发** → **云函数**。
2. 找到并右键 **admin_initDB** → **云端测试**（或「上传并部署」后再测）。
3. 在测试参数中填写：
   ```json
   { "action": "clean_dirty" }
   ```
4. 点击**运行**。返回结果示例：
   ```json
   {
     "success": true,
     "message": "已清空废弃的 papers 集合（mock_bank、real_papers 未动）",
     "deleted": { "papers": 条数 }
   }
   ```

## 可选：清空所有试卷相关集合

若希望同时清空 **mock_bank** 和 **real_papers**（清空后需重新导入模拟题/真题），使用：

```json
{ "action": "clean_dirty", "onlyPapers": false }
```

**注意**：`onlyPapers: false` 会删除模拟题库和真题列表中的所有数据，请确认后再执行。
